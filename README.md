# Obsidian Border Theme - Phycat 重构版 🎨

这是一份基于 **Border** 主题，并融合了 **Phycat** 深度视觉逻辑的“极致定制版”Obsidian 主题。它通过对底层 CSS 的重构，解决了传统美化主题在编辑模式（Live Preview）下的诸多顽疾。

> [!NOTE]
> **主要目标**：在不牺牲任何编辑器（CM6）稳定性的前提下，提供最顶级的“苹果风格”和“高度几何化”的笔记体验。

---

## ✨ 核心视觉特性 (Core Features)

### 1. 深度定制的标题系统 (Header System)
每一级标题（H1-H6）都经过了单独的几何建模，旨在通过呼吸感和辅助图形区分层级。
*   **H1 (居中模式)**：底部带渐变动画的宽幅下划线，悬停时自动展开。
*   **H2 (双子塔)**：左侧双垂直棒修饰，极致对齐文字中轴线。
*   **H3-H6 (几何序列)**：依次采用 **垂直棒、实心圆、空心圆、短横线** 进行前缀修饰，右侧则配有同色同步的 **Mac 层级圆点**（大型 Mask 矢量图标）。

### 2. Mac 窗口化代码块 (Mac Style Codeblocks)
*   **跨模式一致性**：无论是“实时预览”还是“阅读视图”，均展现出完美的 Mac 窗口顶栏（红黄绿三色圆点）。
*   **坐标零漂移**：彻底修复了传统美化导致的光标选区偏移（Selection Bug），通过 `padding-top: 34px` 安全避让方案，确保编辑时 100% 灵敏。
*   **语言标识胶囊**：程序语言（如 Java, Python）标识进行了右偏好避让，视觉极度清爽。

### 3. 行内代码气泡特效 (Inline Code Glow)
普通文本中的 `inline code` 加入了轻量的浮空发光动效。当鼠标悬停时，代码块会自动变色并产生微弱的阴影，提升交互的成就感。

### 4. 玻璃质感表格 (Glassmorphism Tables)
表格加入了磨砂玻璃背板背景，并对表头、表项以及鼠标悬停行进行了深度的对比度调整。

## 🛠️ 安装与使用 (Installation)

1.  **备份**：请先备份你当前的 `theme.css`（位于 `.obsidian/themes/Border/` 路径下）。
2.  **替换**：将本项目中的 `theme.css` 覆盖到上述路径。
3.  **刷新**：在 Obsidian 设置中重新应用主题，即可即时可见所有特效。

> [!TIP]
> 推荐在你的仓库根目录下创建 `Test_Effects.md` 并填入测试文本，以实时预览所有 Hn 级标题以及代码块的渲染效果。
